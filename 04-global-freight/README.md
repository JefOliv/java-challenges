## English Version

### 📝 Scenario & Goal
This project represents the first **Consolidator Challenge (Master Challenge)** of my Java learning journey at Alura. The main goal was to unify and deeply practice all the core concepts explored across the previous challenges (01, 02, and 03) into a single, integrated international maritime freight billing system.

The application processes raw incoming cargo weights, calculates container distribution using type conversion, computes financial operations with strict operator precedence, and generates a structured international invoice.

### 🛠️ Unified Concepts Practiced
* **Primitive Types & Conventions (Challenge 01):** Strict use of `int`, `double`, and `String` objects following the industry-standard `camelCase` naming convention.
* **Text Blocks & Precedence (Challenge 02):** Implementation of multi-line terminal outputs using Text Blocks (`"""`) and isolating high-priority mathematical operations using parentheses `()`.
* **Advanced Formatting (Challenge 02):** Precise control over text placeholders and currency/weight decimal places using `System.out.printf`.
* **Explicit Casting (Challenge 03):** Manual conversion from `double` to `int` to truncate decimals and safely retrieve whole units for logistical allocation.
* **Technical Documentation (Challenge 03):** Demonstrating clean code habits by documenting logic and constraints using single-line (`//`) and multi-line (`/* */`) comments.

## Versão em Português

### 📝 Cenário & Objetivo
Este projeto representa o primeiro **Desafio Consolidador (Master Challenge)** da minha jornada em Java pela Alura. O objetivo foi unificar e aplicar de forma integrada todos os conceitos explorados nos desafios anteriores (01, 02 e 03), simulando um sistema de faturamento de fretes marítimos internacionais. 

A aplicação recebe o peso bruto de uma carga, calcula a distribuição exata em containers utilizando conversão de tipos, processa os custos financeiros com regras de precedência e gera uma fatura internacional detalhada.

### 🛠️ Conceitos Unificados e Praticados
* **Tipos Primitivos & Convenções (Desafio 01):** Uso estrito de `int`, `double` e objetos `String` seguindo o padrão de nomenclatura de mercado `camelCase`.
* **Blocos de Texto & Precedência (Desafio 02):** Implementação de estruturas complexas em múltiplas linhas com Text Blocks (`"""`), garantindo o isolamento de operações matemáticas prioritárias usando parênteses `()`.
* **Formatação Cirúrgica (Desafio 02):** Manipulação precisa de exibição de dados e casas decimais para valores monetários e pesos utilizando `System.out.printf`.
* **Casting Explícito (Desafio 03):** Conversão manual e intencional de `double` para `int` para truncar valores decimais e obter a contagem exata de unidades inteiras de containers.
* **Documentação Técnica (Desafio 03):** Prática de boas maneiras de engenharia de software através do uso coordenado de comentários em bloco (`/* */`) e comentários de linha (`//`).
