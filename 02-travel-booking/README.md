## English Version

### 📝 Scenario & Goal
This project simulates a backend feature for an international travel booking agency. The main objective was to calculate the average daily cost of a trip (ensuring proper operator precedence) and dynamically generate a well-formatted confirmation email using modern text blocks.

### ⚙️ Requirements
- Ensure mathematical operator precedence using parentheses `()` to calculate average costs.
- Utilize **Text Blocks** (`"""`) available since Java 15 for multi-line text formatting.
- Implement advanced string formatting via `System.out.printf` to handle currency decimal places (`%.2f`).
- Include proper code documentation using both single-line (`//`) and multi-line (`/* ... */`) comments.

### 🛠️ Core Concepts Practiced
- **Operator Precedence:** Correctly nesting arithmetic calculations.
- **Advanced Text Manipulation:** Working with the `String` class and multi-line blocks.
- **Formatted Output:** Replacing old string concatenation with structured formatting placeholders (`%s`, `%d`, `%f`).

## Versão em Português

### 📝 Cenário & Objetivo
Este projeto simula uma funcionalidade no backend de uma agência internacional de viagens. O objetivo principal foi realizar o cálculo do preço médio diário de um pacote turístico (garantindo a ordem correta das operações matemáticas) e gerar um e-mail de confirmação elegante e formatado em múltiplas linhas utilizando blocos de texto modernos.

### ⚙️ Requisitos do Desafio
- Garantir a precedência de operadores matemáticos usando parênteses `()` para calcular a média diária.
- Utilizar o recurso de **Text Blocks** (`"""`), disponível a partir do Java 15, para formatação em múltiplas linhas.
- Implementar formatação avançada de textos via `System.out.printf` para controlar as casas decimais de valores financeiros (`%.2f`).
- Praticar a documentação de código adicionando comentários de linha única (`//`) e comentários em bloco (`/* ... */`).

### 🛠️ Conceitos Principais Praticados
- **Precedência de Operadores:** Agrupamento correto de cálculos aritméticos.
- **Manipulação de Texto Avançada:** Uso prático da classe `String` com blocos multilinha.
- **Saída Formatada:** Substituição da concatenação tradicional (`+`) pelo uso profissional de marcadores de posição (*placeholders* como `%s`, `%d`, `%f`).
