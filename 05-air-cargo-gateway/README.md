## Português

### 📝 Descrição do Projeto
Este é um **desafio gerado pela Inteligência Artificial**, projetado para consolidar na prática os aprendizados e conceitos estudados na plataforma da Alura. O objetivo foi aplicar as estruturas condicionais (`if/else`) em conjunto com a estrutura de seleção múltipla (`switch case`) e operadores lógicos para simular o backend de uma esteira de triagem automática de bagagens em um aeroporto internacional.

O sistema analisa a classe do bilhete do passageiro para definir sua franquia permitida e avalia o peso real da bagagem para decidir se o embarque é gratuito, taxado ou se exige despacho obrigatório.

### 🛠️ Competências Praticadas
* **Estrutura de Seleção Múltipla (`switch case`):** Utilizada para mapear a categoria do bilhete às regras exatas de franquia de peso.
* **Lógica Condicional Eficiente:** Uso de `if/else` sem redundâncias desnecessárias, priorizando a legibilidade e desempenho do compilador.
* **Saída Formatada Modernizada:** Uso de `Text Blocks` (`"""`) e `printf` para formatação precisa de casas decimais monetárias.

## English

### 📝 Project Description
This is an **AI-generated challenge**, designed to practically consolidate the learning and concepts studied on Alura's platform. The goal was to combine conditional structures (`if/else`) with multiple-selection control flow (`switch case`) and logical operators to simulate the backend of an automated baggage triage system at an international airport.

The system analyzes the passenger's ticket class to determine their free baggage allowance, and measures the actual baggage weight to decide if the boarding is free, taxed, or if it requires mandatory cargo dispatch.

### 🛠️ Key Skills Applied
* **Multiple-Selection Structure (`switch case`):** Used to map the ticket category to exact weight allowance limits.
* **Efficient Conditional Logic:** Application of logical `if/else` hierarchies without redundancies, optimizing compiler performance.
* **Modernized Formatted Output:** Use of multi-line `Text Blocks` (`"""`) and `printf` to ensure precise formatting of financial decimal places.
